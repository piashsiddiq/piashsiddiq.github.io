# Abu Bakkar Siddiq — Portfolio Website

Personal portfolio and CV site for **Abu Bakkar Siddiq** — CPA Candidate and Dual MS student in Accounting Analytics and Business Analytics & Artificial Intelligence at The University of Texas at Dallas.

🔗 **Live site:** [piashsiddiq.github.io](https://piashsiddiq.github.io)

## About This Site

This site showcases my professional background in financial reporting, ERP systems, and business analytics, including:

- Professional experience across accounting, finance, and MIS roles
- Education (Dual MS, MBS, BBS) and technical skills (SAP S/4HANA, Power BI, Python, R, SQL)
- Academic project work and a publication currently under review
- Downloadable CV (auto-built from LaTeX via GitHub Actions)
- Contact information and links to professional profiles

## Built With

- [Jekyll](https://jekyllrb.com/) static site generator
- [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) remote theme
- Hosted on [GitHub Pages](https://pages.github.com/)
- CV PDF compiled automatically from LaTeX (`cv/main.tex`) via GitHub Actions on every push to `cv/`

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.
